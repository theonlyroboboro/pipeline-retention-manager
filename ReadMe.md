# Pipeline Retention Manager

## Building package

prerequisite

```
npx tfx-cli extension create
```

### Building dev package:

```
tfx extension create --manifest-globs vss-extension-dev.json

```

### Building package:

```
tfx extension create --manifest-globs vss-extension.json

```

## Contributing

Contributions are welcome! If you'd like to contribute, please open an issue to discuss your idea first, then submit a pull request with clear commit messages and a description of the change. For changes to the extension manifest or tasks, confirm compatibility with the Azure DevOps extension tooling and add any required documentation updates.
